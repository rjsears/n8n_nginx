# Storyboard B: Modern Minimalist (Dark Theme)

## Design Philosophy
Same minimalist approach as Storyboard A, but optimized for dark mode to reduce eye strain during extended use. Features elegant dark backgrounds with vibrant accent colors and subtle glowing effects on status indicators.

## Layout Structure

### Navigation
- **Type**: Horizontal top navigation bar
- **Height**: 64px
- **Background**: slate-900 with subtle bottom border (slate-800)
- **Logo**: Left-aligned with icon + text
- **Nav Items**: Center-aligned horizontal list
- **User Menu**: Right-aligned dropdown

### Main Content Area
- **Max Width**: 1280px (max-w-7xl)
- **Padding**: px-4 sm:px-6 lg:px-8
- **Background**: slate-950

### Content Spacing
- **Section Margin Bottom**: mb-6 (24px)
- **Card Gaps**: gap-6 (24px)
- **Element Spacing**: space-y-4 (16px)

## Responsive Behavior

### Mobile (< 640px)
```
Navigation:
- Hamburger menu (top-left) - slate-700 bg
- Logo centered
- User menu (top-right)
- Menu overlay: bg-slate-900/95 backdrop-blur

Dashboard:
- Container cards: 1 column, stacked
- Full-width cards
- Reduced padding (p-4)
- Smaller text sizes

Tables:
- Convert to card view
- bg-slate-900 cards
- Show essential data only
- Tap to expand details

Modals:
- Full screen overlay
- bg-slate-900
- Slide up animation
```

### Tablet (640px - 1024px)
```
Navigation:
- Full horizontal nav
- May compress spacing

Dashboard:
- Container cards: 2 columns
- Charts stack vertically
- Maintains visual hierarchy

Tables:
- Horizontal scroll
- Sticky first column (bg-slate-900)
- Actions menu instead of inline
```

### Desktop (> 1024px)
```
All layouts as designed
Full feature set
Glow effects enabled
Hover animations
Tooltips on all icons
```

## Accessibility Features

### High Contrast in Dark Mode
```
All text exceeds WCAG 2.1 AA in dark mode:
- slate-100 on slate-950: 17.2:1 (AAA)
- slate-400 on slate-950: 6.8:1 (AA Large)
- sky-400 on slate-950: 8.2:1 (AA)
- Status colors enhanced for dark backgrounds
```

### Focus States
```
All interactive elements:
- ring-2 ring-sky-400 ring-offset-2 ring-offset-slate-950
- High contrast against dark backgrounds
- Visible glow effect
```

### Screen Reader Support
```
Same as light theme:
- Semantic HTML
- ARIA labels
- ARIA live regions
- Alt text
- Role attributes
```

## Implementation Notes

### Dark Mode CSS
```css
/* Use Tailwind's dark mode class strategy */
html.dark {
  color-scheme: dark;
}

/* Smooth theme transitions */
* {
  transition: background-color 200ms, border-color 200ms;
}
```

### Performance
```
- Use will-change for animated elements
- GPU-accelerated transforms
- Debounce glow effects
- Lazy load charts
- Optimize shadow filters
```

### Browser Compatibility
```
- Glow effects: Supported in modern browsers
- Fallback: Standard shadows for older browsers
- backdrop-filter: Supported (with -webkit- prefix)
- Gradient text: Supported with fallback
```
