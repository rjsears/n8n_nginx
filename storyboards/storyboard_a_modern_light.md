# Storyboard A: Modern Minimalist (Light Theme)

## Design Philosophy
Clean, spacious, and focused on essential information with generous whitespace. This design prioritizes clarity and ease of use with a horizontal navigation approach. Perfect for users who prefer a traditional, uncluttered interface.

## Layout Structure

### Navigation
- **Type**: Horizontal top navigation bar
- **Height**: 64px
- **Background**: white with subtle bottom border (gray-200)
- **Logo**: Left-aligned with icon + text
- **Nav Items**: Center-aligned horizontal list
- **User Menu**: Right-aligned dropdown

### Main Content Area
- **Max Width**: 1280px (max-w-7xl)
- **Padding**: px-4 sm:px-6 lg:px-8
- **Background**: gray-50

### Content Spacing
- **Section Margin Bottom**: mb-6 (24px)
- **Card Gaps**: gap-6 (24px)
- **Element Spacing**: space-y-4 (16px)

## Responsive Behavior

### Mobile (< 640px)
```
Navigation:
- Hamburger menu (top-left)
- Logo centered
- User menu (top-right)

Dashboard:
- Container cards: 1 column, stacked
- System Health: Full width
- Recent Backups: Full width, below health
- Quick Actions: Stacked buttons, full width

Tables:
- Convert to card view
- Show essential columns only
- Tap row to expand details

Modals:
- Full screen overlay
- Slide up animation
```

### Tablet (640px - 1024px)
```
Navigation:
- Full horizontal nav
- May wrap to two lines if needed

Dashboard:
- Container cards: 2 columns
- System Health: 60% width
- Recent Backups: 40% width
- Maintains grid layout

Tables:
- Horizontal scroll for extra columns
- Sticky first column
- Actions menu instead of inline buttons
```

### Desktop (> 1024px)
```
All layouts as designed above
Full feature set visible
No horizontal scrolling
Hover states enabled
Tooltips on all icons
```

## Accessibility Features

### Keyboard Navigation
```
Tab Order:      Logical flow through interactive elements
Focus Visible:  ring-2 ring-sky-500 on all focusable elements
Skip Links:     "Skip to main content" link at top
Escape Key:     Close modals, dropdowns
Arrow Keys:     Navigate dropdown menus
Enter/Space:    Activate buttons, checkboxes
```

### Screen Reader Support
```
Semantic HTML:      nav, main, aside, footer elements
ARIA Labels:        All icon-only buttons
ARIA Live:          Status updates, toasts
ARIA Expanded:      Dropdown states
Alt Text:           All decorative images (empty alt)
Role Attributes:    Menu, dialog, alert roles
```

### Color Contrast
```
All text meets WCAG 2.1 AA:
- gray-900 on white: 19.41:1 (AAA)
- gray-500 on white: 4.56:1 (AA)
- sky-600 on white: 4.52:1 (AA)
- Status indicators: Never rely on color alone (include icons/text)
```

## Implementation Notes

### CSS Framework
```
Tailwind CSS 3.x with JIT mode
Custom color palette via tailwind.config.js
Inter font from Google Fonts
Heroicons for iconography
```

### State Management
```
Vue 3 Composition API
Pinia for global state
Real-time updates via polling (5s interval)
Optimistic UI updates for immediate feedback
```

### Performance
```
Lazy load routes
Virtual scrolling for long tables
Debounced search inputs (300ms)
Throttled scroll events
Memoized computed properties
```

### Browser Support
```
Modern browsers (last 2 versions)
Chrome, Firefox, Safari, Edge
No IE11 support
Mobile Safari, Chrome Android
```
